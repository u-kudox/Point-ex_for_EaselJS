# Point-ex for EaselJS

Point-ex for EaselJS is extension of EaselJS's Point class.

## Example
	var point1 = new createjs.Point(50, 50);
	point1.normalize(1);
	console.log(point1.toString()); // [Point (x=0.7071067811865476 y=0.7071067811865476)]
